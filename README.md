# Exp - 3 RAILWAY MANAGEMENT SYSTEM

# AIM:
To design and implement a Railway Management System that manages train schedules, passenger details, and ticket bookings efficiently through a computerized interface.
# SRS (Procedure):

#Explanation
1.User Authentication: Verify user credentials for admin and passenger access
2.Train Management: Add, update, or delete train details like train number, name, source, and destination.
3.Ticket Booking & Cancellation: Allow passengers to book or cancel tickets and generate booking details.
4.Status Display: Show train availability, passenger list, and booking status for user reference.
